Book-Movie-Recommender

A Recommender-System made for a HackWeek conducted in my college. A user gives the books name as an input and the Recommender outputs a Movie based on the cosine similarity between the summaries of the Book and the Movie. It’s not suoer accurate as there are a lot of stop words that need to be removed and filters that need to be added. However, I don’t think I’ll work on it because I don’t have the time to.
